# リラックスパズル - 画像ファイル

このフォルダには、リラックスパズルゲームで使用する画像ファイルを配置します。
実際のアプリでは、以下のような画像を用意してください：

## 必要な画像ファイル
1. daysky.jpg - 昼の空の画像（青空と雲）
2. sunset.jpg - 夕暮れの空の画像（夕焼けと雲）
3. nightsky.jpg - 夜空の画像（星空）

## 推奨サイズ
すべての画像は最低でも600×600ピクセル以上、理想的には1200×1200ピクセル以上の解像度があるとより美しく表示されます。
また、パズルとして使用するため、中央に主要な被写体があるような構図が好ましいです。

## 著作権について
商用利用する場合は、著作権フリーの画像を使用するか、適切なライセンスを取得した画像を使用してください。
