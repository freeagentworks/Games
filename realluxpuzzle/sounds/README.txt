# リラックスパズル - 音楽ファイル

このフォルダには、リラックスパズルゲームで使用する音楽ファイルを配置します。
実際のアプリでは、以下のような音楽ファイルを用意してください：

## 必要な音楽ファイル
1. piano.mp3 - 穏やかなピアノ曲（リラックスできる静かな曲調）
2. nature.mp3 - 自然の風音（風の音、鳥のさえずりなど）
3. cabin.mp3 - 機内の環境音（飛行機のエンジン音、機内のアンビエント音）

## 推奨形式と長さ
- ファイル形式：MP3形式（128kbps以上）
- 長さ：最低2分以上、理想的には3〜5分程度のループ再生可能な曲
- ボリューム：適度な音量で、突然の大きな音は避けてください

## 著作権について
商用利用する場合は、著作権フリーの音楽を使用するか、適切なライセンスを取得した音楽を使用してください。
無料で使用できるBGM素材サイトやクリエイティブコモンズライセンスの音源も検討してください。
